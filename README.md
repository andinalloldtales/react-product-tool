# React Product Tool

A full stack product management app built with React, Tailwind CSS, and Axios. Supports creating, editing, deleting, and searching products with real-time feedback.

## Features

- View all products in a responsive grid
- Create, edit, and delete products
- Real-time search by product name
- Low stock and out of stock badges
- Image URL preview on create/edit
- Toast notifications for all actions
- Loading spinner on data fetch

## Tech Stack

- React + Vite
- Tailwind CSS
- Axios
- React Router
- React Hot Toast

## Running locally

1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
